# Custom Interpreter Project 
It is an interpreter written from scratch in C. Version 0.2.0 is the latest and it is live. 
Features,syntax and other information are specified inside each version's repository.

# Why Use/Explore v0.2.0 ?
- The Codebase is split into Makefile,scripts,includes making it easier to navigate the codebase
- Contains clear Documentation and Setup Guide
- The previous versions were rapid prototypes
